# JB우리캐피탈 - 신용대출(4대가입)

> 4대가입 직장인만 진행가능
> (재직전화 아예 안들어감, 재직전화 불편한 고객들 타겟으로 잡으면 좋은 상품)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입직장인 |
| 연령 | 만 23세이상 ~ 만 62세이하 |
| 재직기간 | 3개월이상(3고지2납부 가능)<br>단, 일용직 일 경우 12개월이상 |
| 연소득 | 최소 연봉 제한없음 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만원 ~ 최대 5000만원 |
| 금리 | 최저 11.3% ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장72개월<br>(금액 및 등급별 정해져있어 고객마다 다름) |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 2일, 8일 중 선택 가능 |
| 조회시 DSR확인 | 조회시 DSR확인 된 결과임 |
