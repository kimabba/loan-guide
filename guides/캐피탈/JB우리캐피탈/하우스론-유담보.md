# JB우리캐피탈 - 하우스론(유담보)

> 등록일자 기준으로 3개월 이상된 개인 사업자만 가능
> ★비거주 불가★

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 개인사업자<br>(등록일자 기준) |
| 연령 | 만 20세 이상 ~ 만 65세 이하 |
| 재직기간 | 3개월이상 |
| 연소득 | 최저 소득기준은 없음(매출은 확인되어야 함) |

| 항목 | 내용 |
|------|------|

| 항목 | 내용 |
