# NH농협캐피탈 - 신용대출(프리랜서)

> 법인 회사 이상의 3.3% 소득 공제 프리랜서
> (통장 소득 6개월로 진행되나, 원본 등기 발송 필수)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득공제 프리랜서<br>(회사규모 법인이상 사업체) |
| 연령 | 만 20세이상 ~ 만 65세 이하 |
| 재직기간 | 12개월이상 (통장입금내역 6개월 - 원본등기필수) |
| 연소득 | 2400만 이상<br>(모든 업무 형태 동일) |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 200만 ~ 최대 3500만원 |
| 금리 | 최저 10.9%  ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 84개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 1일, 5일 중 선택가능 |
| 조회시 DSR확인 | 조회 시 DSR적용된 결과임 |
| 가조회 건수제한 | 대략 일주일 이내 당사 포함 3건이상 조회 시 유의심사로 인한 진행 및 심사에 추가시간 소요 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>단, 자동승인 고객 면제 가능 (조회시 확인가능)<br>2. 대표번호 : 회사내 모든 전화번호 가능<br>(단, 검색시 다른쪽에 등록되어있으면 불가)<br>3. 업체서치 : 농캐 전산에서 확인되어야함<br>4. 미등록회사 : 사업자등록증 첨부 후 농캐전산 등록되면 진행가능 |
| 재직확인면제 | 자동승인 고객 면제<br>단, 심사부에서 확인할수 있음 |
| 업체검색 | 필수 (농캐전산에서 서치되어야함) |
| 미등록회사 | 사업자등록증 첨부 후 농캐전산 등록 후 진행가능 |
