# 한국투자캐피탈 - 신용대출(4대가입)

> 4대보험가입 직장인만 진행가능
> (재직전화 아예 안들어감, 재직전화 불편한 고객들 타겟으로 잡으면 좋은 상품)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세 이상 ~ 만 65세 이하 |
| 재직기간 | 6개월이상 |
| 연소득 | 2800만원 이상 |

|------|------|
