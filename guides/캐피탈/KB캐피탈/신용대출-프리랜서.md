# KB캐피탈 - 신용대출(프리랜서)

> 3.3% 소득공제 되는 프리랜서 접수
> (소득자료 꼭 원천징수 아니더라도, 급여통장 6개월로 대체 진행 가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득공제 프리랜서<br>(다단계 불가) |
| 연령 | 만 23세 이상 ~ 만 75세 이하<br>(단, 70세 이상은 한도 거의 안나옴) |
| 재직기간 | 6개월이상 (급여통장 6개월 or 거주자의사업소득원천진수영수증) |
| 연소득 | 보험설계사 : 3000만원이상<br>그외프리랜서 : 2000만원이상 |
| 대환정보 | 사업자 보유자는 대환 대출 불가<br>※ 대환대출 대상<br>KB캐피탈 최종 금리보다 높고,<br>6개월 이상 사용한 대출만 대환 대출 가능<br>*캐피탈, 저축은행, 카드론, 현금서비스 대상<br>*금융사 1곳만 대환 대출 가능<br>(※카드론의 경우 당사보다 금리 낮아도 진행가능)<br>※ 대환시 필요서류<br>- 금융거래 확인서<br>- 대환 정보 캡쳐본<br>- 완납증명서 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 400만원 ~ 최대 1억원<br>※ 약정 기간에 따라 한도, 금리 변동 있음<br>- 진행 전 확인 필수 |
| 금리 | 최저 9.9% ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 96개월<br>(조회 시 차주 별 상환 기간 확인 가능 - 진행 금액 상관없이 최장으로 진행 가능)<br>ex) 한도 4000만원 96개월 가능 → 진행 금액 500만원 96개월 가능 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 거치 후 원리금균등분할상환<br>(조회 시 차주별 거치 가능기간 확인가능) |
| 결제일자 | 1일, 5일, 10일, 15일 중 선택가능 |
| 조회시 DSR확인 | 조회 시 DSR확인된 결과임 |
