# KB캐피탈 - 신용대출(개인사업자)

> 개인사업자, 공동사업자 진행 가능
> (자산 없는 경우 부가세 4800만 이상, 자산 있는 경우 부가세 1000만 이상 접수 가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 개인사업자<br>(공동사업자 진행가능)<br>KCB 1~4등급(751점 이상)<br>신청금액 1000만원 이상 |
| 연령 | 만 30세 이상 ~ 만 75세 이하<br>(단, 70세 이상은 한도 거의 안나옴) |
| 재직기간 | 사업개시 12개월이상<br>부가가치세 과세표준증명원 1년치자료 첨부 필수 |
| 연소득 | 일반 부가세 4800 이상<br>자산 집 차 있는 경우 부가세 1000만원 이상(부가가치세 과세표준증명원상)<br>★로드뷰 심사항목 삭제 단, 매출1,000만원 이상 충족 필수 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 400만원 ~ 최대1억원<br>※ 약정 기간에 따라 한도, 금리 변동 있음<br>- 진행 전 확인 필수 |
| 금리 | 최저 9.9% ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 96개월<br>(조회 시 차주 별 상환 기간 확인 가능 - 진행 금액 상관없이 최장으로 진행 가능)<br>ex) 한도 4000만원 96개월 가능 → 진행 금액 500만원 96개월 가능 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 거치 후 원리금균등분할상환<br>(조회 시 차주별 거치 가능기간 확인가능) |
| 결제일자 | 1일, 5일, 10일, 15일 중 선택가능 |
| 조회시 DSR확인 | 사업자는 DSR적용대상 아님 |
