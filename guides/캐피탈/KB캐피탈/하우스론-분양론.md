# KB캐피탈 - 하우스론(분양론)

> 무직 포함 모든 직군 가능
> (추정소득으로 진행되고, 실소득이 높을 경우 보완해서 진행가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 직업 상관없음<br>NICE 추정소득으로 소득확인 (실제 소득이 높을경우 실소득 반영가능) |
| 연령 | 만 25세 이상 ~ 만 75세 이하<br>(단, 70세 이상은 한도 거의 안나옴) |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 400만 ~ 최대 1억<br>※ 약정 기간에 따라 한도, 금리 변동 있음<br>- 진행 전 확인 필수 |
| 금리 | 최저 4.8% ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 96개월<br>70세이상~80세 미만 : 최단 12개월 ~ 최장 60개월<br>(조회 시 차주 별 상환 기간 확인 가능 - 진행 금액 상관없이 최장으로 진행 가능)<br>ex) 한도 4000만원 96개월 가능 → 진행 금액 500만원 96개월 가능 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 거치 후 원리금균등분할상환<br>(조회 시 차주별 거치 가능기간 확인가능) |
| 만기일시 | 2. 거치 후 원리금균등분할상환<br>(조회 시 차주별 거치 가능기간 확인가능) |
| 결제일자 | 1일, 5일, 10일, 15일 중 선택가능 |
| 조회시 DSR확인 | 조회 시 DSR적용된 결과임 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증, 분양계약서,<br>중도금 납입내역서(금융거래확인서) |
| 임시신분증 / 모바일신분증 | 임시신분증 : 인정가능<br>모바일신분증 : 인정불가 |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 타명의 휴대폰 | 진행가능 |
| 조회인증방식 | 본인명의 휴대폰, 카드인증 |
| 인증서 | 금융인증서, 토스, 패스, 공동인증서, 카카오, 네이버 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인접수 |
| 전자서명 방법 | 1. 카톡이나 문자 URL 링크 접속 후 생년월일 입력<br>2. 고객 정보 입력<br>3. 신분증 촬영<br>4. 약정 전체 동의<br>5. 고객 확인 사항 체크 (예, 근로소득, 가계자금, 원치않음)<br>6. 고객 확인 사항 체크 (1,2,3 아니오)<br>7. 대출 경로 : 광고<br>8. 설명 확인서 체크 (6번만 아니오, 전체 예)<br>9. 신분증 진위 여부 확인 완료 멘트 필수 확인<br>10. 계좌 인증 버튼 클릭<br>11. 해피콜 요청<br>12. 대출 경로 안내 (모두 예 , 수수료 요구만 아니오)<br>13. 대출 내용 확인 후 인증 수단 선택 후 본인 인증<br>14. 약정 전체 체크 후 확인 버튼 |
