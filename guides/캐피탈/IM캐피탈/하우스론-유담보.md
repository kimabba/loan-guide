# IM캐피탈 - 하우스론(유담보)

> 자가 소유 중인 본인, 배우자 사업자 보유시 진행 가능하고, 공동 사업자는 진행 불가
> 집에 근저당 설정되는 담보대출성 상품임(담보대출는 LTV계산이 약간 다름)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 자가소유자중 본인 or 배우자 사업자 보유시 진행가능상품<br>단! 사업자 소유한 사람이 대출차주로  진행됨<br>공동사업자는 진행불가 |
| 연령 | 만 23세이상 ~ 만 65세이하<br>( 남자는 군필이어야 되고 26세 이하는 초본으로 군미필 확인 ) |
| 재직기간 | 사업자등록일 기준 1개월 이상 |
| 연소득 | 부가가치세과세표준증명원 발행 대상자<br>: 무소득 신고자 진행가능<br>부가가치과세표준증명원 발생 미대상자<br>: 면제로 추가 소득자료없이 진행가능 |
| 대환정보 | 대환가능 채무 : 대부, 캐피탈, 저축<br>(단, 사업자 대출의 경우만 대환가능)<br>설정 1건, 한도산출 금액의 90% 이하로 대환<br>(대부 2건 이상, 질권 설정, 전세권 설정 불가)<br>※ 대환시 필요서류<br>- 부채증명서<br>- 대환 정보 캡쳐본 |
