# IM캐피탈 - 하우스론(본인명의)

> 무직 포함 모든 직군 진행가능 ( 즉발 사업자도 진행가능)
> 본인명의, 배우자 공동명의, 타인공동명의 본인지분 50% 이상자!!

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 모든직군 진행가능/무직가능(추정소득) |
| 연령 | 리치론  만 20세 이상 ~ 만 65세이하<br>( 남자는 군필이어야 되고 26세 이하는 초본으로 군미필 확인 ) |
| 즉발사업자 | 즉발사업자 진행 가능<br>(3개월 미만 소득증빙 필수, 3개월이상 소득증빙 생략 가능) |
