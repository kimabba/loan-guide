# 우리금융캐피탈 - 하우스론(분양론)

> 24년 1월 부터 뉴주택(분양권) 상품 폐지
> 무직 포함 모든 직군 가능 ( 추정소득으로 진행되고, 실소득이 높을 경우 보완해서 진행가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 직업 상관없음<br>NICE 추정소득으로 소득확인 (실제 소득이 높을경우 실소득 반영가능) |
| 연령 | 만 25세 ~ 만 65세<br>만25세~29세 or 만60세~64세 한도 축소 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 500만 ~ 최대 1억<br>※ 약정 기간에 따라 한도, 금리 변동 있음<br>- 진행 전 확인 필수 |
| 금리 | 최저 11.7 % ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 84개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 매월 2일만 가능 |
| 추가대출 | 납입 6회차 이상부터 조회시 한도발생되면 추가대출 가능 |
| 조회시 DSR확인 | 조회 시 DSR적용된 결과임 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증, 분양계약서, 중도금 납입내역서, 등본 |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 타명의 휴대폰 | 진행가능 |
| 조회인증방식 | 본인명의 휴대폰, 카드인증 |
| 인증서 | 카카오페이, 패스, 토스 인증서만 가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인 접수 |
| 그외 접수방식 | 1. 지점내방<br>2. 스크래핑(오류 많아서 잘 안함) |
| 내방자서가능지점 | 부산, 광주, 대전 |
| 전자서명 방법 | 1. 전자 약정 안내 문자 내에 링크 클릭<br>2. 전자 서명 클릭<br>3. 고객 정보 입력 후 약관 전체 동의<br>4. 적합성 체크 후 전자 약정 하러가기<br>5. 보이스피싱 예방 체크 후 다음<br>6. 모집경로확인서 확인<br>7. 대출 신청정보 확인후 다음<br>8. 약관 동의 후 확인<br>9. 얼굴 촬영 후 사진 보내기<br>10. 얼굴 인증 완료 멘트 확인 후 다음<br>11. 자동이체 은행, 계좌번호 결제일 선택 후 다음<br>12. 신분증 촬영 후 진위 결과 확인<br>13. 청구정보 확인 및 자택주소 입력 후 다음<br>14. 1원 계좌송금<br>15. 상품설명서 추가확인 및 사설인증 클릭<br>16. 금리인하 요구권 및 대출 철회권 확인 후 다음<br>17. 전자서명 완료 멘트 확인 후 종료 |
