# 롯데캐피탈 - 하우스론(배우자명의)

> 무직 포함 모든 직군 진행 가능
> (신분증 제외 고객준비 서류 없어서 간단하게 진행)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 모든직군 진행가능/무직가능 |
| 연령 | 남자 : 만 26세이상 ~ 만 64세이하<br>여자 : 만 23세이상 ~ 만 64세이하<br>(※ 단, 소득 3개월 이상 증빙 가능 시 만 20세 이상부터 진행 가능) |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만 ~ 최대 6000만원<br>※ 약정 기간에 따라 한도 변동 있음<br>- 진행 전 확인 필수 |
| 금리 | 최저 6.8% ~ 최고 18.9% |
| 상환기간 | 최단 12개월 ~ 최장 120개월<br>(1000만원 미만 진행시 최장 60개월까지 진행가능)<br>※ 전산상 차주 진행 가능한 최장 상환기간 확인  가능<br>→고객별 최장 상환기간내에서 고객 선택 가능 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 원금균등분할상환<br>중 선택가능 |
| 결제일자 | 1일<br>(대출 납부 1회 후 5일 기준으로 변경 가능) |
| 조회시 DSR확인 | 조회 시 DSR적용된 결과임 |
| 가조회 건수제한 | 정해진 조회건수 없음<br>(조회 과다 시 정밀심사 확률 높음,<br>정밀심사 시 부결확률 높음) |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인접수 |
| 전자서명 방법 | ☎?모바일☎?<br>1.{플레이?스토어} 롯데캐피탈 모바일 뱅킹 다운로드<br>2. 상담사번호 + 상담사명 확인<br>3. 이어서 진행 할께요 클릭<br>4. 상담사 코드 번호 입력<br>- 방문판매개시안내사항 확인서 작성대상이십니까?<br>- 아니오/대상이아닙니다  체크!<br>5. 개인정보필수적동의서 전체 동의<br>6. 휴대폰으로 본인 확인 동의하기<br>7. 인증하기<br>☆전자 신청서 작성<br>1. 금액>>기간(더 자세한 조건 알아보기 클릭)<br>2. 원리금 균등>>신청하기 클릭<br>3. 인지세 확인>>선납 적립<br>4. 은행 선택>>계좌 입력>>자동 이체 동의<br>5. 고객 정보 입력<br>6. 직장 정보 입력<br>7. 약관 전체 동의<br>8. 본인 신청 확인 (지인 소개에 꼭 체크)<br>9. 실 소유자 여부(네)<br>10. 대출 신청 정보 확인<br>11. 개인사업자신용대출설명서>>대출 신청하기<br>12. 인증서 선택 - 인증서 비밀 번호 입력하기<br>13. 재직 소득 서류 제출 여부<br>- 네 신청할께요<br>14. 신분증 사진 앞면과 신분증 들고 셀카 찍어서 제출 |
