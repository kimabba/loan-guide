# 유안타저축은행 - 햇살론(4대가입)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세이상 ~ 만 65세이하 |
| 재직기간 | 3개월 이상(3고지 3납부 가능) |
| 연소득 | 1200만원이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 500만 ~ 최대 2000만원 |
| 금리 | 11%내외 |
| 상환기간 | 최단 36개월 , 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 대출 실행일 (1회 납부 후 변경가능) |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |
| 상용직 | 진행가능 |
| 일용직 | 가능 (단, 재직확인 필수 / 급여명세서 필요시 보완) |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 포털사이트나 서치 가능한 번호로 재직확인<br>3. 업체서치 : 네이버, 구글 등<br>(공신력있는 사이트에서 검색되어야 가능)<br>4. 미등록회사 : 진행불가 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 1. 신분증<br>2. 원초본<br>3. 통장갑지 |
| 재직서류 | 1. 건강보험자격득실확인서<br>(다른대처서류없음) |
| 소득서류 | 1. 건강보험납부확인서<br>보완 시 서류리스트<br>1. 국민연금산정용가입내역확인서<br>2. 급여통장입출금거래내역서 |
| 온라인통장 | 진행가능 (토스만 가능) |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | 전자서명 가능수단<br>휴대폰 : 토스 / 패스 / 금융 인증서<br>PC :  공동 인증서 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 스크래핑(전자증명)<br>(신분증, 통장갑지, 초본 첨부 후 본진행) |
| 전자서명 방법 | 1. 전자 약정 링크 접속<br>2. 주민 번호 앞 여섯 자리 클릭<br>3. 전자 계약서 작성하기 클릭<br>4. 인증서 선택 후 동의 체크<br>5. 한도, 금리 확인 후 다음<br>6. 계약 내용 작성 후 내용 확인<br>7. 고객 기본 정보 입력 후<br>8. 약관 및 상품 설명서 전체 동의<br>9. 본인 서명 후 저장<br>10. 파일 인증 누르고, 인증서로 인증 |
