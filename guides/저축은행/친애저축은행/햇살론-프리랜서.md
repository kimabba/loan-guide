# 친애저축은행 - 햇살론(프리랜서)

> 조회시 한도 , 금리 기표 > 정확한 한도는 방문해서 확인 가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득공제 프리랜서 |
| 지역 | □ 프리랜서 : 등본상 거주 지역<br>서울, 충청도, 전라도 지역만 가능 |
| 연령 | 만 20세 이상 ~ 위로 제한 없음 |
| 재직기간 | 6개월이상 |
| 연소득 | 최소 소득 제한은 없으나 소득 발생되는 프리랜서 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 운영 자금 : 최소 500만 ~ 최대 2000만원<br>운영 + 대환 자금 : 최소 500만 ~ 최대 3000만원 |
| 금리 | 최저 9.26% ~ 최고 9.66% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 1일 ~ 25일, 말일 중 선택 가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 재직확인 안함<br>(단, 미등록업체는 진행 불가) |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 1. 신분증<br>2. 등본<br>3. 원초본<br>4. 임대차계약서 (가족명의일시 가족관계증명서)<br>5. 금융거래확인서<br>(기대출 1천만원 이상 중 제일 많은 금융회사 1곳, 채무가 1천만원 미만만 있는 경우 생략)<br>6. 국세, 지방세 완납증명서 |
| 재직서류 | 1. 위촉증명서나 재직증명서<br>(다른대처서류없음) |
| 소득서류 | 1. 거주자의사업소득원천징수영수증<br>2. 거주자의사업소득원천징수영수부<br>3. 지급 월의 소득 및 원천 징수 내용이 표시된 지급명세서 |
| 임시신분증 / 모바일신분증 | 임시신분증 : 인정가능<br>모바일신분증 : 인정불가 |
