# 웰컴저축은행 - 신용대출(4대가입)

> 4대가입 직장인만 진행 가능 (※ 나이, 직군 상관없이 차량소유시에만 신용한도 발생)
> 재직 기간 1개월만 지나도 가능 (고지 1회, 급여 1회, 급여 통장 ars 1개월 필수)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인<br>(※ 나이, 직군 상관없이 차량소유시에만 한도 발생)<br>※ 중금리 상품 : 소득대비 신용채무 100% 이내<br>(순수 신용대출 비율만 인정) |
| 연령 | 만 20세 이상 ~ 만 69세 이하 |
| 재직기간 | 1개월 (고지 1회, 급여 1회 - 급여통장 ars 1개월 필수)<br>※ 중금리 상품 : 3개월 이상 |
| 연소득 | 1200만원 이상<br>※ 중금리 상품 : 3500만원 이상 |
| 대환정보 | 협의후 진행가능 (협의양식 작성)<br>대환 채무나 상세내용 정해진 내용없음<br>※ 중금리 상품 : 대부 사용자 불가<br>※ 대환시 필요서류<br>- 금융거래 확인서 (심사부 요청시)<br>- 대환 정보 캡쳐본<br>- 완납증명서 |
