# 웰컴저축은행 - 햇살론(4대가입)

> 4대가입 직장인만 진행 가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세 이상 ~ 만 69세 이하 |
| 재직기간 | 3개월이상 (3납부 아니면 전직 필수) |
| 연소득 | 최저 소득 제한 없음 (매달 급여 확인 되어야함) |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만 ~ 최대 2000만원 |
| 금리 | 11% 이내 |
| 상환기간 | 최단 36개월 , 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 기본적으로는 급여일당일부터 2일내로 설정 가능 /<br>금일 대출나간다고치면 금일 이후로 4일 뒤로 선택가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |
| 상용직 | 진행가능 |
| 일용직 | 진행가능 (단, 재직확인 필수 / 정밀심사 대상일시 불가) |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 검색되는 회사 대표번호로 재직확인<br>3. 업세서치 : 나이스팁, 아래머니, 크레탑,  로드뷰<br>4. 미등록회사 : 진행가능<br>대표자명함+사업자등록증 받아 확인 (단, 고객마다 상이할 수 있음) |
| 업체검색 | 1. 나이스팁<br>2. 아래머니<br>3. 크레탑<br>4. 로드뷰 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증 |
| 재직서류 | 1. 건강보험득실확인서<br>2. 국민연금가입증명서 |
| 소득서류 | 1. 건강보험납부확인서<br>2. 연금산정용 가입내역 확인서 |
| 임시신분증 / 모바일신분증 | 임시신분증 : 인정가능<br>모바일신분증 : 인정불가 |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | 네이버/카카오페이 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1.공공마이데이터<br>(신분증만 전달가능 - 공마데 불가시 진행불가) |
| 그외 접수방식 | 1. 지점내방(계약서 작성시만 가능) |
| 내방자서가능지점 | 서울 - 용산, 강남, 여의도<br>부산 - 서면<br>경기 - 분당 서현역 ((일산지점 운영안함))<br>인천 - 부평<br>대전 - 둔산동 |
| 전자서명 방법 | WEB 전자계약서 진행<br>1. 첫화면에 전자서명 약정 클릭<br>2. 진행 상품 일괄확인하기<br>3. 전자서명 및 대출계약 체결 완료 클릭<br>4. 전자서명 수단 선택<br>5. 네이버, 카카오 인증하기<br>6. 약관 동의<br>7. 인증서로 인증하기<br>8. 네이버 인증서, 카카오페이로 인증완료<br>9. 인증완료 멘트 확인 후 종료<br>APP 전자계약서 진행<br>1. 웰컴저축 어플 다운로드<br>2. 전자약정 클릭<br>3. 전체 확인 후 전자서명하기 클릭<br>4. 전자서명 수단 선택<br>5. 약관동의 후 인증요청<br>6. 개인정보 3자 동의<br>7. 인증요청 후 다음<br>8. 인증완료 멘트 확인 후 종료 |
