# 웰컴저축은행 - 신용대출(프리랜서)

> 3.3% 소득 공제 프리랜서 진행 가능
> (※ 나이, 직군 상관없이 차량소유시에만 신용한도 발생)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득공제 프리랜서<br>(※ 나이, 직군 상관없이 차량소유시에만 한도 발생) |
| 연령 | 만 20세 이상~ 만 69세 이하 |
| 재직기간 | 1개월 이상( 급여 1회이상 확인) |
| 연소득 | 1200만 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 1억 |
| 금리 | 최저 6.9 ~ 최고 19.9% |
| 상환기간 | 500만 미만 : 2년 / 1000만까지 : 3~4년<br>2000만까지 : 5년 / 2000만 초과 : 7년 (7년이상 되는 고객들 웰컴에 건바이건으로 문의) |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 급여날짜기준 >> 급여일+2일까지 설정<br>단, 매달 29일, 30일, 31일은  설정불가는 동일 |
| 추가대출 | 신용 사용자 > 6개월 이상 납입 후 신용 or 오토론 추가 가능 |
| 조회시 DSR확인 | 조회 시 DSR적용된 결과임 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 검색되는 회사 대표번호로 재직확인<br>3. 업세서치 : 나이스팁, 아래머니, 크레탑,  로드뷰<br>4. 미등록회사 : 진행가능<br>대표자명함+사업자등록증 받아 확인 (단, 고객마다 상이할 수 있음) |
| 업체검색 | 1순위 > 나이스팁 전산 확인<br>2순위 > 아래머니, 크레탑 기업검색사이트 조회<br>3순위 > 로드뷰 등 확인<br>4순위 > 대표자명함+사업자등록증 받아 확인<br>(단, 고객마다 상이할 수 있음) |
| 미등록회사 | 대표자명함+사업자등록증 받아 확인하지만 고객마다 상이할 수 있음 |
