# 웰컴저축은행 - 오토론

> 무직 제외, 소득확인되는 모든 직군 진행가능 (가입자, 미가입자, 프리랜서, 사업자)
> 주부 진행가능 (가족관계 증명서, ~의 배우자로 초본에 확인되어야함)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 무직 제외 소득 확인 되는 모든 직군 가능<br>주부 가능 (가족 관계 증명서, 000의 배우자로 확인되는 초본 필수) |
| 연령 | 만 20세이상 ~ 만 75세이하<br>(만 70~ 75세는 접수해봐야 알수있음) |
| 재직기간 | 1개월 이상(1개월이상 소득증빙자)<br>주부는 상관없음 |
| 연소득 | 연소득 제한없이 진행가능 |
| 대환정보 | - 대환 불가 금융사 : 개인설정<br>※ 대환시 필요서류<br>- 금융거래 확인서 (심사부 요청시)<br>- 대환 정보 캡쳐본<br>- 완납증명서 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만원 ~ 최대1억원<br>(5000만원 이상 나간 경우가 없음) |
| 금리 | 최저 11.4% ~ 최고 19.9% |
| 상환기간 | 1000만원 이하 최단 12개월 ~ 최장 36개월<br>2000만원 이하 최단 12개월 ~ 최장  48개월<br>2000만원 초과 최단 12개월 ~ 최장  60개월<br>※ 직장인+오토의 경우 최장 72개월 가능<br>(금액 상관없음) |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 심사 진행시 심사부와 협의해서 선택 |
| 추가대출 | 진행가능 |
| 추가대출 (차량 2대) | 차량 2대 동시 진행 가능 (2구좌 송금) |
| 조회시 DSR확인 | 조회 시 DSR적용된 결과임<br>(사업자 DSR 미적용) |

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 검색되는 회사 대표번호로 재직확인<br>3. 업세서치 : 나이스팁, 아래머니, 크레탑,  로드뷰<br>4. 미등록회사 : 진행가능<br>대표자명함+사업자등록증 받아 확인 (단, 고객마다 상이할 수 있음) |
| 업체검색 | 1순위 > 나이스팁 전산 확인<br>2순위 > 아래머니, 크레탑 기업검색사이트 조회<br>3순위 > 로드뷰 등 확인<br>4순위 > 대표자명함+사업자등록증 받아 확인<br>(단, 고객마다 상이할 수 있음) |
| 미등록회사 | 대표자명함+사업자등록증 받아 확인하지만 고객마다 상이할 수 있음 |
