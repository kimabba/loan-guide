# 예가람저축은행 - 사잇돌(4대가입)

> 4대가입 직장인, 법인대표 진행 가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인, 법인대표 |
| 연령 | 여성 만 20세 ~ 만 69세 / 남성 만 25세 ~ 만 69세 |
| 재직기간 | 5개월 이상 |
| 연소득 | 1200만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만원 ~ 최대 3천만원 |
| 금리 | 최저 15.2% ~ 최고 18.7% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 대출 실행일이 결제일 |

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 심사자재량<br>2. 대표번호 : 대표번호로만 진행가능<br>3. 서치기관 : 카카오맵, 네이버, 크레탑 등<br>4. 미등록 업체 : 진행불가 |
