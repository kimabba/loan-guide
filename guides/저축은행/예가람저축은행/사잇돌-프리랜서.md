# 예가람저축은행 - 사잇돌(프리랜서)

> 3.3% 소득공제 프리랜서

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득공제 프리랜서 |
| 연령 | 여성 만 20세 ~ 만 69세 / 남성 만 25세 ~ 만 69세 |
| 재직기간 | 5개월 이상 |
| 연소득 | 1200만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만원 ~ 최대 3천만원 |
| 금리 | 최저 15.2% ~ 최고 18.7% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 대출 실행일이 결제일 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 대표번호로만 진행가능<br>3. 서치기관 : 카카오맵, 네이버, 크레탑 등<br>4. 미등록 업체 : 진행불가 |
| 업체검색 | 1. 네이버<br>2. 크레탑<br>3. 카카오맵 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 1. 신분증<br>2. 원초본 |
| 재직서류 | 1. 위촉증명서<br>(다른대체서류없음) |
| 소득서류 | 1. 거주자의사업소득원천징수영수증<br>(월별 또는 연말정산용)<br>※ 본인 세대주일 경우<br>1. 건강보험 납부확인서<br>2. 연금정산용가입확인내역서<br>대체 진행 가능 |
| 임시신분증 / 모바일신분증 | 임시신분증 : 인정불가<br>모바일신분증 : 인정가능 |
