# 스마트저축은행 - 오토론

> 무직 제외, 모든직군 진행가능  (단, 무직의 경우 프리랜서로 단도리 후 진행 가능)
> 최근 1년 신용카드, 체크 카드 사용 내역으로 대체 해서 진행 가능!

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 무직 제외 모든 직군 가능<br>무직의 경우 프리랜서로 단도리 (카드 내역 없는 무직 승인 사례 있음)<br>무직은 최근 1년 카드 사용(신용카드, 체크 카드)  연말 정산 자료 첨부되면 프리랜서로 단도리 후 진행 가능 |
| 연령 | 만 20세이상 ~ 만 70세이하 |
| 재직기간 | 최소 재직기간 없음 상황별 첨부가능 소득자료로 대처하여 진행됨 |
| 연소득 | 최소 연봉 제한없음<br>상황별 소득자료 or 카드사용내역으로 대처하여 진행 |
| 대환정보 | - 대환 불가 금융사 : 개인설정<br>※ 대환시 필요서류<br>- 금융거래 확인서<br>- 대환 정보 캡쳐본<br>- 완납증명서 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만원 ~ 최대 7000만원 |
| 금리 | 최저 15.5% ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 심사 진행중 심사부와 협의 진행됨 |
| 추가대출 | 진행가능 |
| 추가대출 (차량 2대) | 동일차종으로 한도가 남아있으면 진행가능<br>다른 차량으로도 접수가능<br>승인 이후 익일부터 가능 |
| 조회시 DSR확인 | 불가능(심사 시 확인가능)<br>(사업자 DSR 미적용) |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1.온라인접수<br>2. 스크래칭(기표상 DSR대상일시) |
| 전자서명 방법 | 1. 스마트저축 어플 다운로드<br>2. 첫 화면 다음 클릭 후 약관 동의<br>3. 휴대폰으로 본인 인증<br>4. 본인인증 약관 전체 동의 후<br>5. 인증번호 입력 후 다음<br>6. 사용자 인증 수단 선택(지문, 간편비번, 패턴)<br>7. 위 인증수단 중 한가지 선택해 로그인<br>8. 휴대폰 본인인증 한번 더 진행<br>9. 적합성, 적정성 확인 후 다음<br>10. 서류제축 클릭<br>11. 간편서류 제출 - 서류등록 클릭<br>12. 서류 등록 클릭 후 약관동의<br>13. 초본상 주소지 입력 필수<br>14. 인증서 선택 후 인증<br>15. 보완 숫자 입력 후 서류 제출 단계 진행<br>16. 신분증 촬영 후 확인<br>17. 차량 사진 촬영 후 등록<br>18. 전자서명 클릭<br>19. 진행금액, 금리 확인 후 계약서 작성<br>20. 전자서명 약관 전체 동의 클릭<br>21. 대출금액 및 계좌 입력<br>22. 대출 계약서 확인 후 다음<br>23. 인증서 클릭 후 인증<br>24. 계약서 작성 완료 멘트 확인 후 종료 |
