# 삼호저축은행 - 오토론

> 무직 제외, 모든 직군 진행 가능 (단, 무직의 경우 프리랜서로 단도리 후 진행 가능)
> 최근 1년 신용카드, 체크 카드 사용 내역으로 대체 해서 진행 가능!

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 차량 소유한 모든 직군 가능/무직은 불가 - 프리랜서로 단도리<br>무직은 최근 1년 카드사용(신용카드, 체크카드) 연말정산자료 첨부되면 프리랜서로 단도리 후 진행가능 |
| 연령 | 만 19세 이상 ~ 만 65세 이하 |
| 재직기간 | 최소 재직기간 없음 상황별 첨부가능 소득자료로 대처하여 진행됨<br>사업자 6개월 이상 |
| 연소득 | 최소 연봉 제한없음<br>상황별 소득자료 or 카드사용내역으로 대처하여 진행 |
| 대환정보 | - 대환 불가 금융사 : 개인설정<br>※ 대환시 필요서류<br>- 금융거래 확인서<br>- 대환 정보 캡쳐본<br>- 완납증명서 |
