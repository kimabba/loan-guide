# 삼호저축은행 - 신용대출(청년론)

> 군필 / 고등학교 졸업생 / 대학재직자 불가
> 가직장 재직확인 (휴대폰가능) + 소득확인서로 진행가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 군필! 누구든 재직전화만 받아줄 수 있으면 진행가능<br>(휴대폰재직전화 상관없음) |
| 연령 | 만 19세 이상 ~ 만 35세 이하 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만원 ~ 최대 500만원<br>※통합한도 1000만 이내 |
| 금리 | 최저 14.8% ~ 최고 19.95% |
| 상환기간 | 36개월(12개월 단위 연장, 최장 60개월까지 연장 가능) |
| 상환방식 | 만기일시 |
| 만기일시 | 진행가능 |
| 결제일자 | 1일(1일~21일 승인자)<br>15일(22일~말일 승인자) |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 컬러신분증, 원초본 , 최종학력증명서, 통장사본, 소득확인서<br>300만원 이하 진행건 소득확인서 미징구로 진행 가능<br>※단, 대출용도 : 병원비, 생계자금(생활자금) 으로 인지<br>※대출금 용도 확인 및 기존 소득확인서 작성 내용은 고객과 유선상 확인예정<br>※다른 용도로 이야기하면 (ex학원비등) 소득확인서 징구됩니다. |
