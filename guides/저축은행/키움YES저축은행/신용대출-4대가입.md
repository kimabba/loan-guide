# 키움YES저축은행 - 신용대출(4대가입)

> 4대가입 직장인만 진행 가능
> (금리 인하, 한도 증액 관련해서 예외 신청 가능 - 굿포인트 있고, 진행 의사 있는 분만 요청)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세 이상 ~ 만 61세 이하 |
| 재직기간 | 3개월이상 (3고지3납부 가능)<br>※3고지2납일시 신용 좋으면 심사받아봐야 함 |
| 연소득 | 2000만원 이상 |
| 대환정보 | 대환 가능 상품 : 대부, 캐피탈, 저축은행, 카드론<br>대환 불가 상품 : 1금융권, 기타담보, 차량할부, 정부보증상품, 마이너스통장, 현금서비스<br>(단, 현금서비스는 협의 조건으로 예외 가능)<br>※ 계열사 키움저축 대환 가능<br>조회시 대환 가능 차주는 대환 가능사 기표로 남겨줌<br>- 대환 미발생시 협의요청 가능<br>(연소득3천이상/ 재직2년이상 / 자가있을경우 대환 협의 요청 or 신용이력이 우량하면 요청)<br>※ 대환시 필요 서류<br>- 대환 정보 캡쳐본<br>- 대환 후 남은자금 송금이라 완납증 필요없음 |
