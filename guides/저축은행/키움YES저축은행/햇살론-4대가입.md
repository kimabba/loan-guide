# 키움YES저축은행 - 햇살론(4대가입)

> 4대가입 직장인만 진행가능
> (3고지 3납부 진행가능 - 통장 보완 불가)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세이상 ~ 만 54세이하 |
| 재직기간 | 3개월 이상(3고지3납부) |
| 연소득 | 720만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만원 ~ 최대 2000만원 |
| 금리 | 11% 내외 |
| 상환기간 | 최단 36개월 , 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 직접 선택가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |
| 상용직 | 진행가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 심사자 재량 / 햇살론은 거의 안함<br>2. 대표번호 : 대표번호가 원칙<br>(단, 대표자명함, 본인명함상 유선전화로 대체해볼수는 있음)<br>3. 업체서치 : 네이버, 구글<br>(공신력있는 사이트에서 검색되어야 함)<br>4. 미등록업체 : 진행불가 |
| 재직확인면제 | 심사자 재량 / 햇살론은 거의 안함 |
| 업체검색 | 1. 네이버<br>2. 구글<br>(공신력있는 사이트에서 검색되어야 함) |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증 |
| 재직서류 | 1. 건강보험자격득실확인서<br>(다른대처서류없음) |
| 소득서류 | 1. 건강보험납부확인서<br>(다른대처서류없음) |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | 3000만원 미만 - 토스/ 공동/카카오<br>3000이상 이상 - 공동인증서만가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 스크래핑+온라인접수<br>2가지 다 접수해야 진행가능 |
| 전자서명 방법 | 1. 키움예스 어플 다운<br>2. 전자약정 클릭<br>3. 간편 비밀번호 6자리 입력<br>4. 대출계약서 작성 가능 멘트 확인 후 다음<br>5. 진행할 대출 상품 선택<br>6. 한도, 금리 확인 후 다음<br>7. 보이스피싱 문진 내용 확인<br>8. 계약체결 중요사항 안내여부 확인<br>9. 이메일 입력<br>10. 전자 약정 관련 약관 확인 후 동의<br>11. 금리 인하 요구권 안내 확인 후 다음<br>12. 상품설명서 확인<br>13. 팝업 창 확인 후 예 클릭<br>14. 인증서 선택<br>15. 인증서 비밀번호 입력<br>16. 이메일로 전자약정서 발송<br>17. 전자약정이 완료되었습니다 멘트 확인 후 종료 |
