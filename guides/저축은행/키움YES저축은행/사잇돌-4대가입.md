# 키움YES저축은행 - 사잇돌(4대가입)

> 4대가입 직장인, 법인대표 진행 가능
> (전직 인정 가능, 5고지 5납부만 진행 가능, 통장 보완 불가)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인, 법인대표 |
| 연령 | 만 20세이상 ~ 만 61세이하 |
| 재직기간 | 5개월이상 (5고지 5납부) |
| 전직장인정 기준 | 전직 현직 합산 5개월 / 전직2개월이상 / 휴직 90일이내 / 현직 3개월이상 |
| 연소득 | 1200만 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만원 ~ 최대 3000만원 |
| 금리 | 최저 13.51% ~ 최고 18.55% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 직접 선택가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |
| 상용직 | 진행가능 |
| 법인대표 | 진행가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 심사자 재량<br>2. 대표번호 :  대표번호가 원칙<br>(단, 그외 유선번호 요청해볼수 있지만, 심사자 재량)<br>3. 서치기관 : 금융사 자체등록 업체+ 네이버114등록업체<br>4. 미등록회사 : 진행불가 |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | 3000만원 미만 - 토스/ 공동/카카오<br>3000이상 이상 - 공동인증서만가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 스크래핑+온라인접수<br>2가지 다 접수해야 진행가능 |
| 전자서명 방법 | 1. 키움예스 어플 다운<br>2. 전자약정 클릭<br>3. 간편 비밀번호 6자리 입력<br>4. 대출계약서 작성 가능 멘트 확인 후 다음<br>5. 진행할 대출 상품 선택<br>6. 한도, 금리 확인 후 다음<br>7. 보이스피싱 문진 내용 확인<br>8. 계약체결 중요사항 안내여부 확인<br>9. 이메일 입력<br>10. 전자 약정 관련 약관 확인 후 동의<br>11. 금리 인하 요구권 안내 확인 후 다음<br>12. 상품설명서 확인<br>13. 팝업 창 확인 후 예 클릭<br>14. 인증서 선택<br>15. 인증서 비밀번호 입력<br>16. 이메일로 전자약정서 발송<br>17. 전자약정이 완료되었습니다 멘트 확인 후 종료 |
