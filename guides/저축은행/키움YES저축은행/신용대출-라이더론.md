# 키움YES저축은행 - 신용대출(라이더론)

> 1개월 이상 정산 받은 내역  통장으로 확인되는 라이더 진행 상품
> 배달대행 근무자 접수!

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 최근 1개월 이상 정산을 받은 배달대행을 하고 있는 라이더 |
| 연령 | 만 21세 ~ 만 40세 |
| 재직기간 | 1개월 이상 (1개월 이상 정산 받은 내역 확인 필수) |
| 연소득 | 최근 월 100만원 이상 / 연 800만 이상<br>(거래내역상 소득만 인정) |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만 ~ 최대 2500만 |
| 금리 | 최저 9.9% ~  최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할방식 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 재직확인 안함<br>2. 대표번호 : 업체 써치된 대표번호로 유선전화 작성<br>그 외 다른 요청 번호로 재직 확인 불가<br>3. 서치기관 : 금융사 자체등록 업체 + 네이버114등록업체<br>4. 미등록회사 : 진행불가 |
| 업체검색 | 1. 금융사 자체 등록 업체<br>2. 네이버 114 등록 업체 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증, 원초본, 배달대행어플 정상내역 캡쳐본<br>배달대행어플 본인정보 확인되는 화면 캡쳐본<br>통장거래내역 3개월 |

| 항목 | 내용 |
|------|------|

| 인증서 | 3000만원 미만 - 토스/ 공동/카카오<br>3000이상 이상 - 공동인증서만가능 |
