# 삼일저축은행 - 햇살론(미가입)

> 4대보험 미가입 직장인
> (소득 신고 안들어가는 고정급 받는 고객)
> ※ 강원 / 경부 / 대구 거주자 및 재직자만 진행가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대미가입 직장인 |
| 지역 | 취급 지역  [ 강원 / 경북 / 대구 ]<br>- 거주자 : 초본 상 해당 지역 거주자로 전입일은 무관.<br>- 재직자 : 해당 지역 내 회사에 근무하는 재직자로 기업정보 검색 혹은 기업정보 확인 불가시 재직증명서로 근무지 증빙<br>- 채무건수 무관<br>- 채무 불이행 해제자 별도 확인 없이 진행 가능 |
| 연령 | 만 20세이상 ~ 만 60세이하 |
| 재직기간 | 3개월이상(통장3개월 소득확인) |
| 연소득 | 1200만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만 ~ 최대 2000만원 |
| 금리 | 11%내외 (변동금리) |
| 상환기간 | 최단 36개월, 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 대출 실행일 (1회 납부 후 변경 가능) |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 포털사이트나 서치 가능한 번호로 재직확인 (인터넷 번호 불가)<br>3. 업체서치 : 네이버, 구글 등<br>(공신력있는 사이트에서 검색되어야 가능)<br>4. 미등록회사 : 진행불가 |
| 업체검색 | 1. 네이버<br>2. 구글<br>(공신력있는 사이트에서 검색되어야 가능) |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 1. 신분증<br>2. 원초본 |
| 재직서류 | 1. 재직증명서<br>2. 건강보험자격득실확인서(미가입자 확인용)<br>3. 사업자등록증 사본<br>모두 필요 |
| 소득서류 | 1. 급여통장 입출금거래내역서(최근3개월)<br>2. 급여명세서<br>모두필요 |
| 임시신분증 / 모바일신분증 | 임시신분증 : 인정가능<br>모바일신분증 : 인정불가 |
