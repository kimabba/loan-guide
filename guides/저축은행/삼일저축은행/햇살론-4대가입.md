# 삼일저축은행 - 햇살론(4대가입)

> 4대가입 직장인만 진행 가능
> (전직 인정 가능 - 단, 전 직장 통장까지 확인 함)
> ※ 강원 / 경북 / 대구 지역 거주자 및 재직자만 진행가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 지역 | 취급 지역  [ 강원 / 경북 / 대구 ]<br>- 거주자 : 초본 상 해당 지역 거주자로 전입일은 무관.<br>- 재직자 : 해당 지역 내 회사에 근무하는 재직자로 기업정보 검색 혹은 기업정보 확인 불가시 재직증명서로 근무지 증빙<br>- 채무건수 무관<br>- 채무 불이행 해제자 별도 확인 없이 진행 가능 |
| 연령 | 만 20세 이상 ~ 만 60세 이하 |
| 재직기간 | 3개월 이상 (3고지 2납부 가능 - 통장보완) |
| 전직장인정 기준 | 최근 1년내 전직장 + 현직장 합산 3개월<br>(단, 현직장 2고지 1납부 필수)<br>※ 전 직장까지 급여 통장 확인 함 |
| 연소득 | 1200만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만 ~ 최대 2000만 |
| 금리 | 11%내외 (변동금리) |
| 상환기간 | 최단 36개월, 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 대출 실행일 (1회 납부 후 변경 가능) |
