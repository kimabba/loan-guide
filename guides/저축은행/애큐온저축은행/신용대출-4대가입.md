# 애큐온저축은행 - 신용대출(4대가입)

> 4대가입 직장인 , 법인대표 진행가능
> (* DSR오바 되더라도 신규or대환 한도 발생만 되면 중 한가지 상품으로 자동 진행 가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험 가입 직장인, 법인대표 |
| 연령 | 만 24세 이상 ~ 만 64세 이하 |
| 재직기간 | 3개월 이상(3고지 2납부가능) |
| 전직장인정 기준 | 전직 6개월이상 +휴직 텀없음(당일가입) + 현직 1개월이상 |
| 연소득 | 2000만원 이상 |
| 대환정보 | 대환론 한도 발생시 가능<br>대환 가능 채무 - 카드론, 캐피탈, 저축은행<br>대환 불가 채무 - 현금서비스, 대부업, 1금융, 지급보증, 담보대출<br>(금융사가 아닌 계좌수로 최대 5건까지만 대환가능)<br>※ 대환시 필요 서류<br>- 앱진행시 완납 금액 + 가상 계좌 고객 직접 입력 |
