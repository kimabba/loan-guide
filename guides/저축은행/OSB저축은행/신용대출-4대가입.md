# OSB저축은행 - 신용대출(4대가입)

> 4대가입 직장인, 법인대표, 본인사업장에 4대가입한 사업자

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인,사업자,법인대표<br>※ 사업자일경우 직원 포함 4대가입 되어있을때  본인꺼만 득납 따로 발급 제출가능 |
| 연령 | 만 19세이상 ~ 만 59세이하<br>(단, 만 55세 이상은 최대 5년 가능) |
| 재직기간 | 2개월 이상(1고지 1납부 + 급여 1회 수령 필수) |
| 연소득 | 1200만원 이상 |
