# 다올저축은행 - 사잇돌(4대가입)

> 4대가입 직장인, 법인대표 진행가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인, 법인대표 |
| 연령 | 만 20세 이상 ~ 만 59세 이하 |
| 재직기간 | 5개월이상 (5고지 4납부 가능) |
| 전직장인정 기준 | 전직 현직 합산 5개월 / 전직 2개월이상 / 휴직 10일이내 / 현직 3개월이상 |
| 연소득 | 1200만원 |

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 3000만원 |
| 금리 | 최저 8.7% ~ 최고 18.8% |
| 상환기간 | 최단 24개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 결제일자 심사시 확인가능 |
