# 다올저축은행 - 신용대출(프리랜서)

> 3.3% 소득공제 프리랜서
> (주급인정, 급여 나눠서 받는거 전부 합산해서 진행가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득공제 프리랜서 |
| 연령 | 만 20세이상 ~ 제한없음<br>단, 인증서 및 휴대폰 사용 미숙하면 부결될수 있음 |
| 재직기간 | 6개월이상<br>(재직증명서 상 6개월 확인 / 급여 최근 3회 수령 가능) |
| 연소득 | 1000만 이상 |
| 주급/일급 | 진행가능<br>(급여 합산으로 연소득 산정) |

| 결제일자 | 결제일자 심사시 확인가능 |

| 항목 | 내용 |
