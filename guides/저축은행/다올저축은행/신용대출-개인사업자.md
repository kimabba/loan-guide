# 다올저축은행 - 신용대출(개인사업자)

> 개인사업자, 공동사업자 가능
> (공동 사업자 경우 지분 50% 이상일시 가능, 사업자 통장 3개월 필수)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 개인사업자<br>공동사업자 가능 (단, 지분 50%이상만 가능) |
| 연령 | 만 20세 이상 ~ 만 59세 이하 |
| 재직기간 | 3개월 이상 |
| 연소득 | 1000만 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 3000만원 |
| 금리 | 최저 5.9% ~ 최고 19.9% |
| 상환기간 | 원리금 : 1~5년, 7년, 10년 약정 가능<br>(대출금액 상관없이 1, 2, 3, 4, 5, 7, 10년 중에 선택가능)<br>만기일시 : 60개월까지 가능 (6, 12, 24, 36, 48, 60개월)<br>※6개월 단위로 선택가능 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 만기일시상환 |
| 만기일시 | 진행가능 |
| 결제일자 | 결제일자 심사시 확인가능 |
| 조회시 DSR확인 | 사업자는 DSR적용대상 아님 |
