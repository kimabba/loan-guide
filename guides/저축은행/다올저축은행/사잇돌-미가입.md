# 다올저축은행 - 사잇돌(미가입)

> 4대보험 미가입 직장인 (소득 신고 안들어가는 고정급 받는 고객)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대미가입 직장인 |
| 연령 | 만 20세 이상 ~ 만 59세 이하 |
| 재직기간 | 5개월이상 (급여통장 5회 확인시) |
| 연소득 | 1200만 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 3000만원 |
| 금리 | 최저 8.7% ~ 최고 18.8% |
| 상환기간 | 최단 24개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 결제일자 심사시 확인가능 |

## 근무형태

| 항목 | 내용 |
|------|------|

| 파견직 | 고정급여 받는 근로소득 미신고자만 가능 |
