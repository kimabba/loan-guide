# 모아상호저축은행 - 햇살론(4대가입)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 30세이상 ~ 만 65세이하 |
| 재직기간 | 3개월이상 (3고지2납부 가능 - 통장보완) |
| 연소득 | 1200만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만 ~ 최대 2000만원 |
| 금리 | 11%이내 |
| 상환기간 | 최단 36개월, 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 직접 선택가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |
| 상용직 | 진행가능 |
| 일용직 | 진행가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 대표번호로만 진행가능<br>3. 업체서치 : 크레탑, 나이스, kcb, 네이버 검색<br>4. 미등록회사 : 진행불가<br>(단, 네이버 플레이스 등록시 대체가능) |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증 (남성 만 31세이하 원초본 필수) |
| 재직서류 | 1. 건강보험자격득실확인서<br>(다른대처서류없음) |
| 소득서류 | 1. 건강보험납부확인서<br>(보완 시 급여통장 입출금거래내역서 ARS본)<br>( 그 외 다른 대처 서류 인정안됨) |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | SB톡톡 : 공동인증서, 금융인증서<br>스크래핑 : 공동인증서만 가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1.온라인접수 (심사부 팩스 다이렉트 접수만 가능) |
| 전자서명 방법 | 상단 ** 대출센터** → ☆ 전자약정 ☆ →<br>주민번호입력 → 공동인증서 인증 →<br>상품명 (  )확인 → 전자약정 '신청하기'→<br>전체동의 → 다음 → 다음 →<br>계좌이체 1원인증 (??하루 5회 제한! 틀리지않도록 주의!!)→<br>거래목적 (급여 및 생활자금) → 거래자금원천 (대출차입)→<br>필수약관 확인 →이메일 주소 입력  →<br>전자약정체결 →  공동인증서 인증 (인증서 비밀번호 입력) 하시면 완료됩니다. |
