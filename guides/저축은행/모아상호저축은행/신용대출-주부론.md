# 모아상호저축은행 - 신용대출(주부론)

> 신용상 문제는 없지만! 다른상품의 조건이 맞지 않아 접수할 상품이 없는 주부! (대출과다, 연체, 최근대출 이런사람은 한도 안나옴)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 배우자가 세대주인 주부만 가능<br>(배우자 건강보험 미납일시 진행불가) |
| 연령 | 만 30세 이상 ~ 만 60세 이하 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 추정소득으로 진행<br>최소 300만 ~ 최대 1000만원<br>(가조회 시 1000만원이상 한도발생되어도 진행불가) |
| 금리 | 최저 15.5% ~ 최고 19.7% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 심사 시 고객선택 가능 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 1. 신분증<br>2. 원초본<br>3. 건강보험 자격득실확인서(직장피부양자 자격확인)<br>4. 건강보험 납부확인서(직장피부양자 자격확인) |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | SB톡톡 : 공동인증서, 금융인증서<br>스크래핑 : 공동인증서만 가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1.온라인접수 |
| 그외 접수방식 | 1. 서류전달가능 |
| 전자서명 방법 | 상단 ** 대출센터** → ☆ 전자약정 ☆ →<br>주민번호입력 → 공동인증서 인증 →<br>상품명 (  )확인 → 전자약정 '신청하기'→<br>전체동의 → 다음 → 다음 →<br>계좌이체 1원인증 (??하루 5회 제한! 틀리지않도록 주의!!)→<br>거래목적 (급여 및 생활자금) → 거래자금원천 (대출차입)→<br>필수약관 확인 →이메일 주소 입력  →<br>전자약정체결 →  공동인증서 인증 (인증서 비밀번호 입력) 하시면 완료됩니다. |
