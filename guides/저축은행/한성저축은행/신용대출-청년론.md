# 한성저축은행 - 신용대출(청년론)

> 군필 / 졸업증명서 미첨부로 중졸, 대학교 재학생도 말안하면 진행가능
> 가직장 재직확인 (휴대폰가능) + 소득확인서로 진행가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 누구든 재직전화만 받아줄 수 있으면 진행가능(휴대폰재직전화 상관없음)<br>※군미필 불가 |
| 연령 | 만 19세 이상 ~ 만 38세 이하 |

| 항목 | 내용 |

| 한도 | 최소 100만원 ~ 최대 500만원<br>※통합한도 1000만 |
| 금리 | 최저 14% ~ 최고 19% |
| 상환기간 | 12개월 ( 최대 60개월까지 연장가능) |
| 상환방식 | 마이너스 통장 계좌 개설로 사용하는 만큼 이자 납부방식 |
| 만기일시 | 진행가능 |
| 결제일자 | 매월 21일 고정 |
