# SBI저축은행 - 햇살론(4대가입)

> 4대가입 직장인만 접수 가능
> (전직인정 가능, 최저 연소득 제한없이 매달 급여확인시 진행가능 - 3고지 2납부 통장 3회 진행)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세이상 ~ 제한없음 |
| 재직기간 | 3개월 이상 (3고지 2납부 가능 - 통장 보완)<br>※통장진행시 급여명세서 필수 |
| 전직장인정 기준 | 최근 1년이내 전직 납부 2회 + 현직 납부1회 or 만근급여1회(단. 꼭 만근 급여만 접수!) |
| 연소득 | 최저 소득 제한 없음 (매달 급여 확인 되어야함) |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 2000만원 |
| 금리 | 11%내외 (변동금리) |
| 상환기간 | 최단 36개월, 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 대출실행일 |
| 추가대출 | 당행 햇살론 기사용자  : 증액조건으로 가능<br>(타행 근로자 햇살론 기사용자 : 진행불가 )<br>햇살론 사용자  > 신용, 사잇돌 불가능<br>신용, 사잇돌 사용자 > 햇살론 가능 |
