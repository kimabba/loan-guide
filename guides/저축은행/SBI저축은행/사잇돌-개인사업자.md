# SBI저축은행 - 사잇돌(개인사업자)

> 개인사업자, 공동사업자 진행가능
> (공동사업자 경우 지분율 확인 후 심사진행 가능 / 재직확인 필수 아니며, 심사부 판단하에 재직확인 함)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 개인사업자, 공동 사업자<br>※ 공동사업자 경우 지분율 확인 후 심사진행 |
| 연령 | 만 20세이상 ~ 제한없음 |
| 재직기간 | 4개월 이상 (최근 12개월 연금 납부 or 소금원) |
| 연소득 | 600만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 3000만원 |
| 금리 | 최저 8.9%  최고 19.9% |
| 상환기간 | 500만원 이하 최단 12개월 ~ 최장 36개월<br>510만원 이상 900만원 이하 최단 12개월 ~ 최장 48개월<br>1000만원 이상 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 원금균등분할상환<br>중 선택가능 |
| 결제일자 | 1일 ~ 25일 중 선택가능 |
| 추가대출 | 햇살론 사용자  > 사잇돌 불가능<br>사잇돌 사용자 > 햇살론 가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 지입차주 | 진행가능 |
| 공동사업자 | 진행가능(지분율 관련해서 심사진행해야함) |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 심사부가 판단하여 심사함/ 유선확인이 안되면 보완서류 요청하며 판단 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증 |
| 재직서류 | 1. 사업자등록증 + 사업자등록증명<br>(다른대처서류없음) |
| 소득서류 | 1. 소득금액증명원<br>2. 국민연금산정용가입내역확인서<br>중 선택가능 |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 본인명의 휴대폰, 금융 & 공동인증서 가능 |
| 인증서 | 공동인증서, 금융인증서 가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인 접수<br>(심사부 팩스 다이렉트 접수만 가능) |
| 전자서명 방법 | 1. 바빌론닷넷 어플 다운(핸드폰)<br>sbi저축은행 홈페이지 접속 (PC)<br>2. 전자서명 클릭<br>3. 인증서 로그인 (공동 / 금융)<br>4. 개인정보 작성 후 전자서명 클릭<br>5. 약관 전체 동의 후 확인<br>6. 인증서로 인증 후 다음<br>7. 대출 상품 및 정보 확인<br>8. 전체 확인 후 다음<br>9. 대출 정보 확인 후 다음(확인)<br>10. 인증서로 인증 후 확인<br>11. 전자서명이 완료되었습니다 멘트 확인 / 완료<br>* 신분증 인식 안되면 링크 들어가시면 영상통화하기 버튼으로 진행가능 |
