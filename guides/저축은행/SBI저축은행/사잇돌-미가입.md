# SBI저축은행 - 사잇돌(미가입)

> 4대보험 미가입 직장인
> (소득 신고 안들어가는 고정급 받는 고객)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험미가입 직장인 |
| 연령 | 만 20세이상 ~ 제한없음 |
| 재직기간 | 5개월 이상 (급여통장 5회) |
| 연소득 | 1200만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 3000만원 |
| 금리 | 최저 8.9% ~ 최고 19.9% |
| 상환기간 | 500만원 이하 최단 12개월 ~ 최장 36개월<br>510만원 이상 900만원 이하 최단 12개월 ~ 최장 48개월<br>1000만원 이상 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 원금균등분할상환<br>중 선택가능 |
| 결제일자 | 1일 ~ 25일 중 선택가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 회사내 유선번호 입력<br>3. 업체서치 : 네이버, 구글<br>4. 미등록회사 : 진행가능하나 부결확률 높음<br>네이버 등록하고 진행가능 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증 |
| 재직서류 | 1. 재직증명서<br>(다른대처서류없음) |
| 소득서류 | 1. 급여통장 입출금거래내역서 ARS본(최대 1년분) + 급여명세서<br>(다른대처서류없음) |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 본인명의 휴대폰, 금융 & 공동인증서 가능 |
| 인증서 | 공동인증서, 금융인증서 가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인 접수<br>(심사부 팩스 다이렉트 접수만 가능) |
| 전자서명 방법 | 1. 바빌론닷넷 어플 다운(핸드폰)<br>sbi저축은행 홈페이지 접속 (PC)<br>2. 전자서명 클릭<br>3. 인증서 로그인 (공동 / 금융)<br>4. 개인정보 작성 후 전자서명 클릭<br>5. 약관 전체 동의 후 확인<br>6. 인증서로 인증 후 다음<br>7. 대출 상품 및 정보 확인<br>8. 전체 확인 후 다음<br>9. 대출 정보 확인 후 다음(확인)<br>10. 인증서로 인증 후 확인<br>11. 전자서명이 완료되었습니다 멘트 확인 / 완료<br>* 신분증 인식 안되면 링크 들어가시면 영상통화하기 버튼으로 진행가능 |

## 소득관련

| 항목 | 내용 |
|------|------|
| 통장급여(계산) | 동일 날짜일시 급여 합산 인정 가능<br>(급여명세서상 추가소득은 인정 가능하나, 급여통장으로는 대체 불가)<br>예시 ) 최근 3개월 급여 100만원씩 수령, 1년에 2회 200만원 상여 수령시 1200만 + 200만으로 1400만원으로 연소득 합산 가능<br>※ 급여명세서로만 인정 가능 |

## 특이사항

| 항목 | 내용 |
|------|------|
| 상품별 동시진행 | 신용대출+대환대출+사잇돌+햇살론 동시진행가능<br>SBI 상품 이용중이면 SBI 햇살론 진행가능<br>SBI저축 햇살론 사용중일때 SBI 다른상품도 접수불가 |
| 타코드 조회이력 | 당일 포함 4영업일 이후 진행가능<br>(본사 취소 불가 - 조회시 조회 가능 일자 확인 가능)<br>▶ 조회 시 【 타채널 】로 확인되는 경우 협의 후 조회 및 진행 가능<br>└ 타코드 협의 불가<br>└ 선조회건 취소시 당일 재조회 가능 (해제 후 조회 가능여부 확인 필요) |
| 심사 부결시 재접수 가능 시기 | 한도발생시 바로 진행가능 |
