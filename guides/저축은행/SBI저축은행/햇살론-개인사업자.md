# SBI저축은행 - 햇살론(개인사업자)

> 가조회시 가능 여부 확인 > 고객 진행 요청 이후 심사자 서류 안내1~2일 소요 >
> 고객 서류 준비하여 지점 방문 > 심사하여 한도와 이자율 확인 가능 > 총 소요 시간 2~4주 가량 걸림

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 개인사업자 |
| 지역 | □ 사업자 : 사업자등록증상 지역<br>서울, 세종, 부산, 경남, 제주, 울산 제외한 전국<br>(경기지역 : 분당 / 수원 / 일산 / 인천(안산, 부천, 광명, 김포, 군포지역) : 인천지점<br>강원/경북지역 : 대구지점<br>광주/전남/전북지역 : 광주지점 (광주,전남지역 프리랜서 6개월미만인 경우 진행불가)<br>충남/충남/대전지역 : 대전지점 |
| 연령 | 만 20세이상 ~ 제한없음 |
| 재직기간 | 3개월 이상 |
| 연소득 | 최소 소득 제한은 없으나 소득 확인되는 개인사업자 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소100만원 ~ 최대 2000만원(대환가능시 최대 3000만)<br>대환은 심사시 확인가능하며 거의 한도발생 안됨 |
| 금리 | 11%내외 (변동금리) |
| 상환기간 | 최단 36개월, 최장 60개월 중 선택가능 (1년거치도 재단심사로 판단) |
| 상환방식 | 1. 원금균등분할상환<br>2. 1년거치 48개월 원금균등분할상환 가능 (단, 재단에서 결정되는 부분으로 고객마다 다름) |
| 결제일자 | 대출실행일 |
| 추가대출 | 당행 햇살론 기사용자  : 증액 여부 심사접수후 확인가능<br>(타행 근로자 햇살론 기사용자 : 진행불가 )<br>햇살론 사용자  > 사잇돌 불가능<br>사잇돌 사용자 > 햇살론 가능 |
