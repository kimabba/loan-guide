# SBI저축은행 - 햇살론(미가입)

> 2026년1월7읿부터 중단

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대미가입 직장인 |
| 연령 | 만 20세이상 ~ 제한없음 |
| 재직기간 | 3개월 이상 (급여 3개월이상 수령시) |
| 연소득 | 최저 소득 제한없음 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 2000만원 |
| 금리 | 11%내외 (변동금리) |
| 상환기간 | 최단 36개월, 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 대출실행일 |
| 추가대출 | 당행 햇살론 기사용자 : 증액조건으로 가능<br>(타행 근로자 햇살론 기사용자 : 진행불가 )<br>햇살론 사용자 > 신용, 사잇돌 불가능<br>신용, 사잇돌 사용자 > 햇살론 가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인접수 (심사부 팩스 다이렉트 접수만 가능) |
| 전자서명 방법 | 1. 바빌론닷넷 어플 다운(핸드폰)<br>sbi저축은행 홈페이지 접속 (PC)<br>2. 전자서명 클릭<br>3. 인증서 로그인 (공동 / 금융)<br>4. 개인정보 작성 후 전자서명 클릭<br>5. 약관 전체 동의 후 확인<br>6. 인증서로 인증 후 다음<br>7. 대출 상품 및 정보 확인<br>8. 전체 확인 후 다음<br>9. 대출 정보 확인 후 다음(확인)<br>10. 인증서로 인증 후 확인<br>11. 전자서명이 완료되었습니다 멘트 확인 / 완료<br>* 신분증 인식 안되면 링크 들어가시면 영상통화하기 버튼으로 진행가능 |
