# SBI저축은행 - 신용대출(미가입)

> 4대보험 미가입 직장인 (소득 신고 안들어가는 고정급 받는 고객)
> (한도 증액, 금리 인하 1% 가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험미가입 직장인 |
| 연령 | 일반상품 : 만 20세이상, 위로 제한없음<br>퍼스트상품 : 만 29세~59세 |
| 재직기간 | 일반상품 : 3개월 이상 (3납부 미만 - 통장3개월 보완가능)<br>퍼스트상품 : 12개월 이상<br>※통장진행시 급여명세서 필수 |
| 연소득 | 최저소득 제한없음(매달 월급은 확인되어야 함) |
| 대환정보 | ※대환 가능 채무 조회시 차주 휴대폰 문자로 전송됨<br>대환가능 금융사 - 카드론, 캐피탈, 저축은행, 현금서비스, 차량할부, 기타담보, 지급보증가능 (단 이자율,상품 단도리 필수// 실행대출 보다 금리 높다고 말해야 함)<br>대환불가 금융사 - 1금융 은행,  보험사, 카드이용대금, 대부 대환 불가<br>※ 대환시 필요 서류<br>- 완납증명서 |

| 항목 | 내용 |
