# IBK저축은행 - 신용대출(4대가입)

> 4대가입 직장인 진행 가능 ( 온라인 대출 신청이라 재직 전화 대부분 면제로 재직 확인 불편한 고객들 접수시 간단하게 진행가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세 이상 ~ 만 60세 이하 |
| 재직기간 | 4개월이상 (3고지 3납부 필수) |
| 연소득 | 2400만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 1억 |
| 금리 | 최저 12.30% ~ 최고 19.99% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할방식 |
| 결제일자 | 5, 10, 15, 20, 15일 중 선택 가능 |
| 추가대출 | 신용 사용자 > 햇살론 접수가능<br>햇살론 사용자 > 신용 접수가능 |
| 조회시 DSR확인 | 조회시 확인 불가 / 심사중 확인 가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능<br>(기본 조건 일반 직장인 상품과 동일<br>-본심사시 부결 및 감액 가능성 있음) |
| 상용직 | 진행가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 재직전화 필수 아님<br>(심사부 판단으로 재직전화 가는 경우도 있음)<br>2. 대표번호 : 업체서치시 확인되는 번호, 검색 안될시 회사내 유선번호로도 가능<br>3. 업체서치 : 업체검색 안되거나 미등록 업체도 진행가능<br>4. 미등록회사 : 진행가능 |
| 재직확인면제 | 재직전화 필수 아님<br>(대출 신청시 유선번호는 입력해야함)<br>※심사부 판단으로 재직전화 가는 경우도 있음 |
| 업체검색 | 검색 안되어도 진행가능 |
| 미등록회사 | 진행가능 |
