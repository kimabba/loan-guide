# IBK저축은행 - 햇살론(프리랜서)

> 가조회시 가능여부 확인>
> 고객 진행요청 이후 심사자 서류안내1~2일소요>
> 고객 서류준비하여 지점방문>
> 심사하여 한도와 이자율 확인가능>
> 총 소요시간 2~4주가량 걸림

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득공제 프리랜서<br>거주자별사업소득원천징수 발행 가능 프리랜서 |
| 지역 | □ 프리랜서 : 등본상 거주 지역<br>부산 동래,진구 지점 |
| 연령 | 만 19세이상 ~ 만 64세이하 |
| 재직기간 | 6개월이상 |
| 연소득 | 최소 소득 제한은 없으나 소득 발생되는 프리랜서 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만원 ~ 최대 2000만원<br>(대환가능시 최대 3000만)<br>대환은 심사시 확인가능하며 거의 한도발생 안됨 |
| 금리 | 연 8.52% ~ 연 8.92% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 1. 원금균등분할상환<br>2. 1년거치 48개월 원금균등분할상환 가능(단, 재단에서 결정되는 부분으로 고객마다 다름) |
| 결제일자 | 대출실행일 |
