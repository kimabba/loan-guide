# 상상인저축은행 - 오토론

> 직장인, 프리랜서만 진행 가능 / 무직, 사업자 진행 불가 (단, 무직과 사업자의 경우 프리랜서로 단도리 후 진행 가능)
> 최근 1년 신용카드, 체크 카드 사용 내역으로 대체 해서 진행 가능!

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 직장인 또는 프리랜서/ 사업자와 무직은 진행 불가<br>사업자 or 무직 - 프리랜서로 단도리<br>최근 1년 카드 사용(신용카드, 체크 카드)  연말 정산 자료 첨부되면 프리랜서로 단도리 후 진행 가능 |
| 연령 | 만 20세이상 ~ 만 65세이하 |
| 재직기간 | 최소 재직기간 없음 상황별 첨부가능 소득자료로 대처하여 진행됨 |
| 연소득 | 최소 연봉 제한없음<br>상황별 소득자료 or 카드사용내역으로 대처하여 진행 |
| 대환정보 | - 대환 불가 금융사 : 개인설정<br>※ 대환시 필요서류<br>- 금융거래 확인서<br>- 대환 정보 캡쳐본<br>- 완납증명서 |
