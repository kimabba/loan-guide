# 고려저축은행 - 신용대출(개인사업자)

> 개인사업자, 공동사업자 진행가능
> (공동 사업자 경우 차주앞으로 사업소득 통장 확인시 지분율 상관없이 진행가능) / 대부사용시 진행 불가

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 개인사업자, 공동사업자 가능<br>※ 차주앞으로 통장소득 확인시 가능 (지분율 상관없음) |

| 대환정보 | ※ 2025.05.13 기준<br>대환상품 중단 (상품 리뉴얼)<br>3개월이상 채무<br>- 대환 가능 채무 : 저축은행, 캐피탈, 카드론만 가능<br>(그 외 채무 불가능)<br>- 저축은행 2건이하 (햇살론 사용시 3건까지),<br>- 캐피탈, 카드론 건수 제한 없음<br>※ 대환시 필요 서류<br>- 필수 서류는 없으나, 심사부에서 금융거래확인서나 완납증명서 요청 할 때 있음 |
