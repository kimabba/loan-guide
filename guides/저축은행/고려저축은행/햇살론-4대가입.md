# 고려저축은행 - 햇살론(4대가입)

> 4대가입 직장인만 진행가능
> (통장소득 필수 확인 / 납부상 문제있을 경우 대체 서류 연금서류만 가능)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세이상 ~ 만 69세 |
| 재직기간 | 3개월이상(3고지 2납부 - 연금 3회일시 진행가능) |
| 연소득 | 1200만원 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 600만 ~ 최대 2000만원 |
| 금리 | 11%내외 |
| 상환기간 | 최단 36개월, 최장 60개월 중 선택가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 대출실행일(1회 납부 후 변경가능) |
| 추가대출 | 당사 기대출 3회차 이상 납부자 한도 발생시 진행 가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>(단, 대기업 공기업 정규직은 출입증, 사원증으로 대체되는 경우 있음)<br>2. 대표번호 : 회사내 유선번호로 가능<br>(인터넷 전화일 경우 정밀심사)<br>3. 업체서치 : 네이버, 구글<br>(공신력있는 사이트에서 검색되어야 가능)<br>4. 미등록회사 : 진행불가 |
| 재직확인면제 | 대기업 공기업 정규직은 출입증, 사원증으로 대체되는 경우 있음 |
| 업체검색 | 1. 네이버<br>2. 구글<br>(공신력있는 사이트에서 검색되어야 가능) |
