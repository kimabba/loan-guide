# JT저축은행 - 신용대출(4대가입)

> 4대가입자만 가능
> (신용대출 평균 금리 12~15%대로 이자율을 타사대비 저렴한편, 단 한도면에서는 조금 부족한편임)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인<br>※즉시대출은 법인업체 이상만 가능<br>※즉시대출 :  남성 만 29세 ~ 만 53세 , 여성 만 24세 ~ 만 53세 |
| 연령 | 만 20세이상 ~ 만55세이하<br>※즉시대출 :  남성 만 29세 ~ 만 53세 , 여성 만 24세 ~ 만 53세 |
| 재직기간 | 3개월이상 (3고지 3납부이상 가능) |
| 연소득 | 2000만원 이상 |
| 대환정보 | 채무 통합론 한도 발생시 진행 가능<br>- 대환 가능 채무 : 캐피탈, 저축, 카드론, 대부, 중고차 할부, 기타 할부(오토론)<br>- 대환 불가 채무 : 은행, 지급 보증, 신차 할부 등<br>(누가 봐도 저금리로 보이면 대환 불가)<br>대환시 건수 제한은 없으나~ 1건이라도 개설 1개월 지난 채무가 있어야 1개월 이전 대출도 대환 가능<br>※ 계열사 jt친애저축 대환 가능<br>※ 대환시 필요 서류<br>- 상환 정보만 전달시 진행 (완납증 요청 X) |
