# 융창저축은행 - 햇살론(4대가입)

> 4대가입 직장인만 진행 가능
> (급여 통장 진행시, 급여 편차 30만원 이상일시 진행 불가)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 20세이상 ~ 만 60세이하 |
| 재직기간 | 3개월이상(3고지2납부- 통장급여 3개월) |
| 전직장인정 기준 | 최근 1년이내 전직 납부 2회 + 현 근무지 납부 1고지 1납 진행 가능<br>※ 납부상 연소득 초과일 경우 급통으로 진행 불가 |
| 연소득 | 1200만원이상 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인 접수 |
| 전자서명 방법 | PC 약정시<br>1. 융창저축 홈페이지 접속후 전자약정 클릭<br>2. 이름, 주민번호 입력 후 공동인증서 인증 틀릭<br>3. 대출정보 확인, 전체 동의 후 확인 클릭<br>4. 이메일 입력 후 확인<br>5. 약정서 요약본 전부 확인 후 클릭<br>6. 개인전보 조회 동의 40페이지 모두 동의 후 전자약정 확인 클릭<br>7. 인증서 선택 후 비밀번호 입력<br>8. 약정 완료 멘트 확인 후 종료<br>SB톡톡플러스 약정시<br>1. sb톡톡 플러스 어플 다운로드<br>2. 하단에 대출신청 / 전자약정 클릭<br>3. 진행 상품 클릭후 전자약정 클릭<br>4. 저축은행 선택에서 융창 저축은행 클릭 후 다음<br>5. 전자약정서 작성 클릭<br>6. 고객 정보 입력 후 인증서 선택<br>7. 인증서 가져오기 진행<br>8. pc-> 스마트폰 인증서 복사<br>9. 인증번호 생성<br>10. pc에서 융창저축은행 홈페이지 접속해 이어서 진행하기<br>11. 인증센터 -> 인증서 복사<br>12. 스마트폰 승인번호 입력, 내보내기 완료 후 다시 휴대폰에서 이어서 진행<br>13. 인증서 비밀번호 입력<br>14. 약관 전체 동의 후 다음<br>15. 동의 및 서명 확인 40페이지 모두 체크<br>16. 전자약정 확인 클릭<br>17. 인증서 비밀번호 한번 더 입력 후 인증완료<br>18. 인증완료 멘트 확인 후 종료 |
