# 융창저축은행 - 사잇돌(4대가입)

> 4대가입 직장인, 법인대표 진행 가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인, 법인대표 |
| 연령 | 만 20세 이상 ~ 만 60세 이하 |
| 재직기간 | 5개월 이상(5고지 5납) |
| 연소득 | 1200만 이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 200만 ~ 최대 3000만 |
| 금리 | 최저 9.9% ~ 최고 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 원리금균등분할상환 |
| 결제일자 | 실행날짜 (1회 납부 후 변경가능) |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |
| 상용직 | 진행가능 |
| 일용직 | 진행가능 |
| 법인대표 | 진행가능 |
| 4대가입 사업자 | 진행가능<br>(4대보험 가입시, 직장인 기준으로 진행 가능) |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 심사자 판단<br>2. 대표번호 : 서치 가능한 번호로 재직 확인<br>(단, 서치불가시 대표자 명함으로 진행가능 - 유선전화 필수)<br>3. 업체서치 : 네이버, 구글 등<br>(공신력있는 사이트에서 검색되어야 가능)<br>4. 미등록회사 : 대표자 명함으로 대체 진행가능 |
| 재직확인면제 | 사대가입 + 유선번호 인터넷 서치시 재직면제 가능 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 신분증 |
| 재직서류 | 1. 건강보험자격득실확인서<br>(다른대처서류없음) |
| 소득서류 | 1. 건강보험자격득실확인서<br>2. 소득금액증명원(작년 만근자 / 전년도 중도 입사자일 경우 근로소득원천징수영수증 추가 첨부) |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | 공동인증서 / 금융인증서 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 1. 온라인 접수<br>(단, 심사부 다이렉트 발송건만 인정) |
| 전자서명 방법 | PC 약정시<br>1. 융창저축 홈페이지 접속후 전자약정 클릭<br>2. 이름, 주민번호 입력 후 공동인증서 인증 틀릭<br>3. 대출정보 확인, 전체 동의 후 확인 클릭<br>4. 이메일 입력 후 확인<br>5. 약정서 요약본 전부 확인 후 클릭<br>6. 개인전보 조회 동의 40페이지 모두 동의 후 전자약정 확인 클릭<br>7. 인증서 선택 후 비밀번호 입력<br>8. 약정 완료 멘트 확인 후 종료<br>SB톡톡플러스 약정시<br>1. sb톡톡 플러스 어플 다운로드<br>2. 하단에 대출신청 / 전자약정 클릭<br>3. 진행 상품 클릭후 전자약정 클릭<br>4. 저축은행 선택에서 융창 저축은행 클릭 후 다음<br>5. 전자약정서 작성 클릭<br>6. 고객 정보 입력 후 인증서 선택<br>7. 인증서 가져오기 진행<br>8. pc-> 스마트폰 인증서 복사<br>9. 인증번호 생성<br>10. pc에서 융창저축은행 홈페이지 접속해 이어서 진행하기<br>11. 인증센터 -> 인증서 복사<br>12. 스마트폰 승인번호 입력, 내보내기 완료 후 다시 휴대폰에서 이어서 진행<br>13. 인증서 비밀번호 입력<br>14. 약관 전체 동의 후 다음<br>15. 동의 및 서명 확인 40페이지 모두 체크<br>16. 전자약정 확인 클릭<br>17. 인증서 비밀번호 한번 더 입력 후 인증완료<br>18. 인증완료 멘트 확인 후 종료 |
