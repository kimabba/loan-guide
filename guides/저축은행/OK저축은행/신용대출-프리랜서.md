# OK저축은행 - 신용대출(프리랜서)

> 6개월 이상 통장에 소득 확인되는 3.3% 소득 공제 프리랜서
> (월급, 주급, 일급 인정되고, 불가 직군 없는 금융사 / 만기일시 진행시 중수료 면제)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 3.3% 소득 공제 프리랜서 |
| 연령 | 만 25세 이상 ~ 만 60세 이하<br>만 61세~ 70세 미만은 1000만원이 최대한도 |
| 재직기간 | 6개월이상 / 통장급여 6개월 이상 확인시 (일급, 주급, 월급) |
| 연소득 | 918만 이상 |
| 주급/일급 | 진행가능<br>(급여 합산으로 연소득 산정) |

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 5000만원<br>(지점에서는 최소금액 500만으로 말함) |

| 상환기간 | 최단 12개월 ~ 최장 120개월<br>만기로 진행시 1년 |
