# OK저축은행 - 하우스론(본인명의)

> 아파트 소유중인 4대가입, 법인대표 진행가능
> 개인설정 있어도 진행가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대가입 직장인, 법인대표 |
| 연령 | 만 25세 이상 ~ 만 60세 이하<br>만 61세~ 70세 미만은 1000만원이 최대한도 |
| 재직기간 | 3개월 이상 (3고지2납부 진행가능) |

| 조회시 DSR확인 | 조회 시 확인안됨/ 심사시 DSR확인하여 심사함 |
