# OK저축은행 - 햇살론(4대가입)

> 4대보험 가입자만 진행가능
> (※통장진행불가)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만 19세이상 ~ 위로 나이 제한없음 |
| 재직기간 | 3개월이상(3고지 3납 필수) |
| 전직장인정 기준 | 현직장 1개월 이상 / 1년이내 전직포함 3개월 이상<br>(현직 납부 1회 이상 필수 - 통장진행시 부결) |
