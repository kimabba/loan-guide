# OK저축은행 - 신용대출(개인사업자)

> 개인사업자, 공동사업자 진행가능
> (소득 1가지라도 확인시 진행 가능/ 공동 사업자 경우 지분율 상관없이 진행 차주 서류로만 진행)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 개인사업자<br>공동사업자 (지분율 상관없음, 진행 차주 서류만 필요) |
| 연령 | 만 25세 이상 ~ 만 60세 이하<br>만 61세~ 70세 미만은 1000만원이 최대한도 |
| 재직기간 | 6개월 이상<br>※ 유의업종 : 음식점, 제조업, 소매업 36개월 이상 |
| 연소득 | 소매업/제조업/음식업<br>업력3년이상+소득2,000만원이상<br>도매업/수리업<br>업력6개월+소득3,000만원이상<br>기타 업종<br>업력6개월+소득1,500만원이상<br>※업종이 곂칠 경우 EX) 도매업 및 소매업 사업자<br>고객에게 유리하게 적용 |
| 즉발사업자 | 지역의보, 지역연금 납부시 진행가능 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만 ~ 최대 5000만원<br>(지점에서는 최소금액 500만으로 말함) |
| 금리 | 최저 5.9% ~ 최고 19.99% |
| 상환기간 | 최단 12개월 ~ 최장 120개월<br>만기로 진행시 1년 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 원금균등분할상환<br>3. 만기일시상환 (단, 500만원까지만 가능)<br>중 선택가능 |
| 만기일시 | 500만원까지 만기일시 진행가능<br>500만원 초과시 불가능 |
| 결제일자 | 3일 간격으로 결제일 지정할수 있음 선택하면됨 |
| 추가대출 | 우수고객의 경우 진행가능<br>(현재 사용고객 조회 전 우수고객인지 확인요청가능 , ) |
| 조회시 DSR확인 | 조회 시 확인안됨/ 심사시 DSR확인하여 심사함<br>(※ 사업자도 직장인과 같이 DSR 적용대상) |
