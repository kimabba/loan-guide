# 한국투자저축은행 - 신용대출(4대가입)

> 4대가입 직장인, 사업자 진행 가능 (사업자 경우 비대면으로만 가능)
> 단기 재직자 진행 가능( 고지 / 납부 없어도 가능 - 1개월 재직 + 급여 통장 1회)

| 항목 | 내용 |
|------|------|

| 연소득 | 1000만원 이상 |
| 대환정보 | 대환가능채무 : 대부,저축은행 및 캐피탈 신용대출,카드론 등 /사잇돌 햇살론17상품에 한해선 금리 확인 후 취급 가능<br>대환불가능채무 :  1금융,현금서비스, 담보대출, 기타담보(신차할부,중고차 담보)<br>대환계좌 확인시 금리 미확인 : 금융거래 확인서 필수<br>심사부 요청시 : 완납증명서 보완<br>대환 진행시 한투보다 금리 낮아도 진행가능<br>(단, 심사부 통화시 금리 높아도 가용 +대환 진행하고싶은데 한건으로 묶고싶다고 답변하면 진행가능)<br>※ 대환 대상이 있을경우 고객이 심사자와 대환하고 싶다 협의요청을 해야 한도증액 가능여부 확인가능<br>협의 불가 라고 상담사에게 전달오는 메신저 기표는 참고만 할것<br>※ 대환시 필요 서류<br>- 고객마다 다르기 때문에 심사 진행해봐야 확인 가능 |
