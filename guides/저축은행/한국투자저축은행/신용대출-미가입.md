# 한국투자저축은행 - 신용대출(미가입)

> 4대보험 미가입 직장인 (소득 신고 안들어가는 고정급 받는 고객)
> 단기 재직자 진행 가능 (30일 이상 근무자 최대한도 2000만원), 자가 소유시 대환 협의 가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험미가입 직장인 |
| 연령 | 신용 : 만 19세 이상 ~ 만 63세이하<br>(단, 만 60세 이상은 정밀심사 대상 + 최대한도 3000만원)<br>※ 가이드라인은 정해져 있지만, 현재는 위로 나이제한없이 한도 나오면 진행되고 있으니 고연령 고객도 접수 진행 가능 |
| 재직기간 | 3개월이상 (통장급여 3회)<br>※단기 재직자 경우 1회 이상 소득 증빙 (건보+통장) 및 재직 확인 必 |
| 연소득 | 1000만원이상 |
| 대환정보 | ● 미가입/프리랜서 자가 소유시 심사시 대환 협의 가능<br>ㆍ자가 소유시 심사자 판단하에 대환조건으로 증액하여 송금 가능 (사전 협의 불가)<br>대환계좌 확인시 금리 미확인 : 금융거래 확인서 필수<br>심사부 요청시 : 완납증명서 보완<br>대환 진행시 한투보다 금리 낮아도 진행가능<br>(단, 심사부 통화시 금리 높아도 가용 +대환 진행하고싶은데 한건으로 묶고싶다고 답변하면 진행가능)<br>※ 대환 대상이 있을경우 고객이 심사자와 대환하고 싶다 협의요청을 해야 한도증액 가능여부 확인가능<br>협의 불가 라고 상담사에게 전달오는 메신저 기표는 참고만 할것<br>※ 대환시 필요 서류<br>- 고객마다 다르기 때문에 심사 진행해봐야 확인 가능 |
