# 우리금융저축은행 - 햇살론(4대가입)

> 4대가입 직장인만 진행 가능
> (전자 서명이 간단해서, 인증서 사용 어려운 고객들 진행 용이 / 단, 수당율 낮아서 필요시에만 진행 요망)

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 4대보험가입 직장인 |
| 연령 | 만20세이상 ~ 제한없음 |
| 재직기간 | 3개월이상(3고지 3납부 가능) |
| 연소득 | 400만원이상 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 300만 ~ 최대 2000만원 |
| 금리 | 11%내외 |
| 상환기간 | 최단 36개월 , 60개월 중 선택 가능 |
| 상환방식 | 원금균등분할상환 |
| 결제일자 | 대출실행일 기준<br>1일 ~ 25일이내 : 대출실행일<br>26일 ~ 말일 : 10일, 15일, 20일 중 선택가능 |

## 근무형태

| 항목 | 내용 |
|------|------|
| 계약직 | 진행가능 |
| 파견직 | 진행가능 |
| 상용직 | 진행가능 |
| 일용직 | 진행가능 |

## 재직확인

| 항목 | 내용 |
|------|------|
| 재직확인방식 | 1. 재직확인 : 필수<br>2. 대표번호 : 회사내 유선번호로 가능<br>(단, 유선번호 없을시 대표자 휴대폰으로 대체 가능)<br>3. 업체서치 : 네이버<br>5. 미등록업체 : 진행불가 |
| 업체검색 | 1. 네이버 |

## 구비서류

| 항목 | 내용 |
|------|------|
| 기본서류 | 1. 신분증<br>2. 원초본<br>3. 통장갑지 |
| 재직서류 | 1. 건강보험자격득실확인서<br>2. 국민연금가입증명서<br>중 선택가능 |
| 소득서류 | 1. 건강보험납부확인서<br>2. 연금산정용가입내역확인서<br>3. 소득금액증명원<br>4. 근로소득원천징수영수증<br>중 선택가능 |
| 온라인통장 | 진행가능 (카카오뱅크, 케이뱅크, 토스 가능) |

## 조회/인증

| 항목 | 내용 |
|------|------|
| 조회인증방식 | 휴대폰인증(본인명의폰)<br>(그 외 인증방식 불가) |
| 인증서 | 간편인증번호 6자리 (처음 인증시 설정)<br>공동인증서 가능 |

## 접수방식

| 항목 | 내용 |
|------|------|
| 메인 접수방식 | 온라인 접수<br>(신분증, 초본 전달가능 / 그 외서류 본사팩스전송 필수) |
| 전자서명 방법 | 1. 전자서명 안내 문자 내 링크 클릭<br>2. 첫페이지 전자계약 클릭<br>3. 휴대폰 본인인증 후 전체 동의 확인<br>4. 전자서명 글자 클릭<br>5. 대출계약서 확인 후 다음<br>6. 고객 확인 사랑 예 -> 확인<br>7. 나머지 단계는 모두 셋팅되어 있어 따로 체크 할거 없이 순서대로 다음 클릭하면 됨<br>8. 인증하기 누르고 비밀번호 6자리 2번 입력<br>9. 1원 송금, 계좌확인 진행<br>10. 전자약정 완료 멘트 확인 후 종료 |
