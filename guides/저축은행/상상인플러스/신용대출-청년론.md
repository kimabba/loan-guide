# 상상인플러스 - 신용대출(청년론)

> 군필 / 고등학교 졸업생 / 대학재직자 불가
> 가직장 재직확인 (휴대폰가능) + 소득확인서로 진행가능

## 기본내용

| 항목 | 내용 |
|------|------|
| 대상 | 누구든 재직전화만 받아줄 수 있으면 진행가능(휴대폰재직전화 상관없음)<br>※군미필 불가 |
| 연령 | 소득확인서  진행<br>- 남 : 만 19세 이상 ~ 만 31세 이하<br>- 여 : 만 19세 이상 ~ 만 29세 이하<br>추정소득 진행<br>- 남 : 만 32세 ~ 만 45세 이하<br>- 여 : 만 30세 ~ 만 45세 이하 |

## 상품정보

| 항목 | 내용 |
|------|------|
| 한도 | 최소 100만원 ~ 최대 500만원<br>※통합한도 1500만 |
| 금리 | 19.9% |
| 상환기간 | 최단 12개월 ~ 최장 60개월 |
| 상환방식 | 1. 원리금균등분할상환<br>2. 만기일시상환 |
| 만기일시 | 진행가능 |
| 결제일자 | 결제일 첫 달 상플에서 지정, 이후 변경 가능 - 1,5,10,21일 |
